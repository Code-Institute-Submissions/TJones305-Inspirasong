How to get Image URL?

* any letters in link need to be lowercase

Getting Image URL

  1. Google Search Images for picture
  2. Click on image
  3. Open webpage of image
  4. Locate image on webpage it's on, right-click and select "copy image address"
  5. Paste image address into Profile Image

NB: A good indicator the address is correct is to check that an image file is at the end of the web address (e.g: .png, .jpg .jpeg)

Here's an example of a good link, it has a png picture format and the link is all lowercase: https://www.easydrawingtips.com/wp-content/uploads/2018/05/female_face_line_drawing.png

If the link has Uppercase letters there is a chance it may not load on your profile correctly
